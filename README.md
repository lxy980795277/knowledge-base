# 个人知识库

用于长期保存和整理个人学习资料、创作素材、项目方案与灵感记录。

## 内容目录

- [`ppt/`](./ppt/)：PPT 相关的视觉素材、封面方案与项目资料。
- [`images-study/`](./images-study/)：图片生成与信息海报的研究案例，包含提示词和创作要点。
- [`cartoon/`](./cartoon/)：卡通、动漫相关的资料与素材。
- [`releases/`](./releases/)：每次 Git 提交对应的版本更新记录。

## 更新记录

本仓库使用规范化的 Git 提交信息，并在每次提交时同步生成一份版本说明。请查看 [`releases/README.md`](./releases/README.md) 了解全部更新与版本规则。

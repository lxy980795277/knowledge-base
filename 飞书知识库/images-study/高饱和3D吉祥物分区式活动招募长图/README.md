# 高饱和3D吉祥物分区式活动招募长图

![](./image-01.png)

## 核心要点

- **用首屏“角色 + 大标题”建立活动情绪**：顶部集中放置立体中文标题、吉祥物和主题道具，让读者先感知活动类型、活力和参与氛围。
- **按“时间—步骤—规则—回报—行动”组织长图**：正文阅读路径与用户决策顺序一致，适合承载活动招募、规则说明和行动入口。
- **让吉祥物承担跨区块导览功能**：同一角色以挥拍、指引、宣布、示范等动作重复出现，把分散模块串成一套叙事。
- **用高饱和外框承托低干扰信息底板**：标题、标签和行动按钮使用强色彩，正文区使用白色纸张、浅网格和柔和阴影，兼顾吸睛与可读性。
- **把复杂信息压缩为可扫描容器**：时间节点、三步流程、奖励卡和底部入口各自拥有固定容器，避免段落漂浮和信息挤压。

## Prompt

```plain text
生成一张约 1:2.07 的中文竖向活动招募长图，主题是一场公司内部羽毛球联赛。整体采用高饱和、明亮、亲和的 3D 吉祥物活动海报风格，信息量大但自上而下的阅读路径必须清楚。

风格与材质：
- 顶部是蓝色天空、半透明运动丝带、草地和立体运动道具，使用橙、绿、蓝、黄色的高光 3D 中文标题。
- 使用原创的圆润企鹅吉祥物作为导览角色，深蓝身体、白色腹部、红色围巾，动作活泼但比例统一。
- 正文使用带轻微纸张纹理的白色大卡片、浅色网格、胶带贴纸、圆角标签、轻柔投影和高饱和描边。
- 图标采用闹钟、日历、清单、礼物、奖杯、球拍、羽毛球和喇叭等立体小物件。

画面结构：
- 顶部约 25%：超大立体主标题，企鹅吉祥物、球拍、羽毛球和运动轨迹围绕标题；标题下有一条蓝色赛事副标题和一条绿色报名提示。
- 第一正文区约 15%：标题“赛程时间”，一条横向时间线包含四个节点，用短标签和日期表达报名组队、抽签、小组赛、决赛与闭幕。
- 第二正文区约 16%：标题“报名步骤”，浅网格底板中横向放置三个步骤卡，分别是意愿组队、确认队长队名、填写报名表。
- 第三正文区约 20%：标题“报名规则”，集中放置四行可扫描说明，配少量绿色手绘下划线与淡色球拍装饰。
- 第四正文区约 13%：标题“赛事奖励”，左右两张并列圆角卡，分别表达团队荣誉与个人荣誉。
- 第五正文区约 5%：绿色渐变行动按钮，文案短、字号大，带手指点击图标。
- 底部约 10%：标题“工作人员招募”，左侧为岗位、时间和地点三行信息，右侧放不可扫码的二维码占位，页脚保留一行联系说明。

排版：
- 中文标题必须粗壮、紧凑、有白色描边和立体阴影；模块标题放在绿色撕纸质感标签上。
- 时间线、三步卡和双奖励卡严格对齐，不让角色遮挡文字。
- 长说明只出现在规则区和底部信息区，其余区块使用短标签。
- 整体保持上部吸睛、下部清晰承载信息的节奏。

约束：
- 角色五官、翅膀、脚和球拍握持必须自然，禁止多余肢体。
- 所有中文文字放在独立容器中，避免挤压、溢出、重叠或被插画遮挡。
- 二维码只做不可扫码的装饰占位，不出现真实网址、水印或第三方品牌标识。
- 不使用写实照片、扁平企业图库插画、暗色背景或低对比文字。
```
